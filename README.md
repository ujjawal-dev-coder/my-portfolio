Ujjawal Sharma | Creative Developer & Designer Portfolio
Yeh mera personal portfolio website hai jise maine modern UI/UX principles aur clean code ka use karke banaya hai. Isme meri coding skills aur graphic design work ka perfect blend dikhaya gaya hai.

✨ Features
Modern Dark UI: Cinematic dark theme ke saath high-contrast blue accent colors.

Fully Responsive: Mobile, tablet aur desktop sabhi screens par perfectly kaam karta hai.

Smooth Animations: Scroll events aur Intersection Observer API ka use karke smooth fade-in effects daale gaye hain.

Bento-style Grid: Projects aur showcase ke liye modern grid layout ka use kiya gaya hai.

Contact Form: FormSubmit integration ke saath working contact section.

🛠️ Tech Stack Used
Frontend: HTML5, CSS3 (Custom Variables, Flexbox, Grid)

Interactivity: Vanilla JavaScript (ES6+)

Icons: FontAwesome 6.4

Fonts: Google Fonts (Inter)

Animations: Intersection Observer API & CSS Transitions
